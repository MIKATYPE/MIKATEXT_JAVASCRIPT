美佳の英文タイプ テキスト練習編 Ｖｅｒ２．０２．０１ JAVASCRIPT版ソースコード

　教育用のタイプ練習ソフト「美佳の英文タイプ テキスト練習編V2.02」のJAVASCRIPT版ソースコードです。
　ブラウザーで作動します。
　Chrome、Edgeで動作確認済です。
　chromebookでも動作します。
　Firefoxでは文字「'」がショートカットキーに使用されているため正常に作動しません。
　著作権を表示する初期画面は表示されません。
　成績はクッキーに保存されます。保存期間は一年です。
　オフラインでも作動しますがオフラインで実行した場合は成績は保存されません。
　Windows版では練習時間記録ファイルmikatext.logと最高速度記録ファイルmikatext.spdが保存されますが、
　このJAVASCRIPT版ではmikatext.logとmikatext.spdは保存されません。
　Windows版では合計練習時間はプログラムを起動してから終了するまでの時間ですがJAVASCRIPT版では個別の累積練習時間の合計です。
　Windows版では実行時にウィンドーのサイズを変更できますが、このJAVASCRIPT版では実行時にサイズの変更はできません。
　Windowsのサイズ変更したい場合はindex.htmlのキャンバスサイズを変更して下さい。
　メニューの項目に「終了」はありません。終了したいときはウィンドーをクローズして下さい。
